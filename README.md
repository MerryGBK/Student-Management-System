# Student Management System (Python)

A simple Student Management System built in Python to manage student records.
The application supports adding, updating, searching, and deleting student data with basic validation.

## Technologies
- Python

## Features
- Add new student records
- Update existing records
- Search students by key fields (e.g., ID / name)
- Delete student records
- Input validation and clean output formatting

## Skills Demonstrated
- Data structures (lists/dictionaries)
- Control flow (conditions, loops)
- Functions and modular code
- Basic error handling

## Purpose
 Academic/practice project demonstrating Python fundamentals and structured program design.
